--- title: Cenos启用网络并安装Rstudio server author: Glidis date: '2017-08-20' slug: cenos-rstudio-server categories: - linux tags: - r studio server ---
  1. 先安装Centos
    Centos Error: Cannot find a valid baseurl for repo: base
    sudo dhclient
  2. 安装R
    sudo yum update
    sudo yum install epel-release
    sudo yum install R
  3. 安装Rstudio server
    可以参考Rstudio官网
    若没有wget命令，先
    sudo yum install wget
    然后
    wget https://download2.rstudio.org/rstudio-server-rhel-1.0.153-x86_64.rpm
    sudo yum install --nogpgcheck rstudio-server-rhel-1.0.153-x86_64.rpm
    systemctl status rstudio-server.service
    然后访问http://yourip:8787。

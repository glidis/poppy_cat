match.call用在function中直接得到function的列表

xxx <- function(a, b, c, d,e){
  mc <- match.call()
  mc
}
xxx(1,2,3)  # 'call' class, `[` and `[[` methods are available
## xxx(a = 1, b = 2, c = 3)
